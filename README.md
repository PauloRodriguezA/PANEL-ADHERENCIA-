# Panel de Adherencia Entel Connect

Panel gerencial publicable con datos ya consolidados.

## Ejecutar local

```bash
pip install -r requirements.txt
streamlit run panel.py
```

La vista abre por defecto en `Todo` para comparar IBM, SAO y ECC.

## Contenido incluido

- Panel Streamlit.
- Bases WFM depuradas IBM, SAO y ECC.
- KPI Disponibilidad y Reclamos finales IBM/SAO.
- KPI Uso correcto de herramienta.
- Bases EPA SQLite de IBM, SAO y ECC.
- Logos y configuracion visual.

No se incluyen PST, caches clasificadas de correos, adjuntos, scripts de extraccion ni archivos temporales.
